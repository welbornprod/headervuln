Header Vulnerability Checker
============================

Checks response headers for accepted values.


Command-line help:
==================

```
Usage:
    headervuln.py -h | -v
    headervuln.py URL

Options:
    URL           : URL to check. Protocol must be specified, otherwise
                    http:// is used.
    -h,--help     : Show this help message.
    -v,--version  : Show version.
```
